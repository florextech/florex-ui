# @florex/ui

A dark-first React component library built with Tailwind CSS. Minimal, accessible, and beautiful.

## Installation

```bash
npm install @florex/ui
```

## Quick Start

```tsx
import { Button, Card, CardHeader, CardTitle, CardContent } from '@florex/ui';

function App() {
  return (
    <Card>
      <CardHeader>
        <CardTitle>Welcome</CardTitle>
      </CardHeader>
      <CardContent>
        <Button>Get Started</Button>
      </CardContent>
    </Card>
  );
}
```
